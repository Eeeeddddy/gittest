checkRep

public void checkRep()
Checks the following well-formedness properties for every statement in the list of statements:
  • The number of inputs is the same as the input of inputs specified by the statement kind.
  • Type type of each input is compatible with the type required by the statement kind.
  • The input variables come from earlier statements.
  • The result variable does not appear in earlier statements.

insert

public Map<MVariable,MVariable> insert(int index,
                                       MSequence seq)
inserts the statements making up the given sequence into this sequence, at the given index. The parameter sequence is left unchanged. The receiver sequence is mutated. For convenience, returns a mapping from variables in the parameter sequence to their corresponding variables in the modified receiver sequence.

Parameters:
index - The index at which to insert the given sequence.
seq - The sequence to insert.
Returns:
A mapping from variables in the parameter sequence to their corresponding variables in this sequence.
